# Bar do Pereira - Interface Base

Projeto base em **React + Vite** para o sistema do Bar do Pereira.

## O que já vem nesta interface

- Dashboard inicial
- Comandas abertas
- Adição de produtos na comanda
- Cadastro manual de produtos
- Ficha de sinuca com valor fixo de **R$ 3,00**
- Fechamento da comanda com:
  - Dinheiro
  - Pix
  - Cartão
  - Dividir
  - Fiado
- Troco apenas para pagamento em dinheiro
- Divisão por pessoa
- Bloco de sinuca com Mesa 1 e Mesa 2
- Cupom de fiado com:
  - Produtos
  - Fichas
  - Pagamentos parciais
  - Valor pendente
- Alertas visuais

## Como rodar

```bash
npm install
npm run dev
```

## Como subir no GitHub

1. Crie um repositório no GitHub
2. Envie estes arquivos
3. Rode `npm install`
4. Rode `npm run dev`
